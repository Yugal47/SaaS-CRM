# SaaS-CRM
A cloud-native CRM where multiple companies (tenants) can:

+ Register their organization
+ Manage customers
+ Manage leads
+ Manage sales pipelines
+ Generate reports
+ Upload documents
+ Receive notifications
+ Use secure authentication
+ Access only their own data

Frontend Requirements consist
+ React
+ Nginx
+ CloudFront
+ S3

Backend Requirements Consist
+ Spring Boot
+ Docker
+ EKS

For Database
+ Postgres SQL - RDS
+ DynamoDB

For Authentication purpose
+ cognito

For Messaging
+ SQS
+ SNS

For File storing
+ S3

And for Monitoring 
+ Prometheus
+ Grafana
+ CloudWatch

Pre-requisites

Terraform
AWS configurations
Docker/ECS
ECS/Kubernetes
